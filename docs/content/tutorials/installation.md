# Installing Terracotta

This page describes how to download and install Terracotta CLI.

## GitHub Releases

You can download compiled native binaries directly from the [Releases](https://github.com/beduality/terracotta/releases) page.

### Linux & macOS

1. Download the binary for your platform.
2. Make it executable:
   ```bash
   chmod +x terracotta
   ```
3. Move it to your path:
   ```bash
   mv terracotta /usr/local/bin/
   ```

### Windows

1. Download the `terracotta-windows-amd64.exe` binary.
2. Add the directory containing the binary to your system PATH environment variable.
