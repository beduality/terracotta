# Repository Documentation

This section is for people who work **on** Terracotta itself: contributors, maintainers, and release managers. If you use Terracotta in your own project, start with the [Core](../modules/core/README.md), [Gradle Plugin](../modules/gradle-plugin/README.md), or [Integration](../integration/README.md) docs instead.

This section follows the [Diátaxis](https://diataxis.fr/) documentation framework:

- **Tutorials** guide new contributors through a complete first contribution.
- **How-To Guides** solve specific maintainer or contributor tasks.
- **Reference** lists facts about the repository, tooling, and conventions.
- **Explanation** discusses why the project is organized the way it is.
