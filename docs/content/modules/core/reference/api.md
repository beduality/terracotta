# Core API Documentation

The public API of `terracotta-core` is documented with KDoc and generated by Dokka. The generated documentation includes package overviews, class signatures, and member-level documentation.

## Generated API reference

- **[Core API Docs](../../../apidocs/terracotta-core/index.html)** — Full KDoc output for `terracotta-core`.

## Building locally

To generate the API docs locally, run:

```bash
./gradlew :terracotta-core:dokkaHtml
```

To generate the combined multi-module docs:

```bash
./gradlew dokkaHtmlMultiModule
```

The MkDocs build copies the generated docs into `docs/apidocs/` automatically via the `copy_kdoc.py` hook.
