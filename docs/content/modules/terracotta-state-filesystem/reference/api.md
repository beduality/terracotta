# State Filesystem API Documentation

## Generated API reference

- **[State Filesystem API Docs](../../../../apidocs/terracotta-state-filesystem/index.html)** — Full KDoc output for `terracotta-state-filesystem`.

## Building locally

To generate the API docs locally, run:

```bash
./gradlew :terracotta-state-filesystem:dokkaHtml
```

To generate the combined multi-module docs:

```bash
JAVA_HOME=/usr/lib/jvm/java-21-openjdk ./gradlew :dokkaHtmlMultiModule --no-daemon
```

The MkDocs build copies the generated docs into `docs/apidocs/` automatically via the `copy_kdoc.py` hook.
