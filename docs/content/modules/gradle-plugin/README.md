# Gradle Plugin

The Terracotta Gradle plugin applies project metadata and artifacts to remote registries from within your build.

This section follows the [Diátaxis](https://diataxis.fr/) documentation framework:

- **Tutorials** guide new users through a complete end-to-end workflow.
- **How-To Guides** solve specific configuration problems.
- **Reference** provides lookup information for tasks and the generated API docs.
- **Explanation** discusses concepts, design decisions, and lifecycle behavior.
