# Modules

Terracotta is organized into focused modules.

## Available modules

- **[Core](core/README.md)** — metadata resolution, diff engine, loaders, conventions, and provider interfaces.
- **[Gradle Plugin](gradle-plugin/README.md)** — `terracotta` Gradle DSL, tasks, and CI/CD integration.
- **[State Filesystem](terracotta-state-filesystem/README.md)** — file-backed state backend that persists run state to a local YAML file.
- **[Modrinth Provider](provider-modrinth/README.md)** — provider implementation for the [Modrinth](https://modrinth.com/) registry.
- **[Hangar Provider](provider-hangar/README.md)** — provider implementation for the [Hangar](https://hangar.papermc.io/) registry.

Use the navigation menu to explore tutorials, how-to guides, reference, and explanation pages for each module.
