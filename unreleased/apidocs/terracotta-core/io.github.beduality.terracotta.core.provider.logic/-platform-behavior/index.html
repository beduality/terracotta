PlatformBehavior

interface PlatformBehavior

Describes how a registry behaves when applying Terracotta operations.

The platform behavior is used by io.github.beduality.terracotta.core.provider.RegistryProvider implementations to decide which operations can be applied. Stateful platforms can update project metadata, while append-only platforms support only file uploads.
