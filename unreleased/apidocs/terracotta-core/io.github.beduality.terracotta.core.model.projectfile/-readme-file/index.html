ReadmeFile

class ReadmeFile(val content: String?, convention: ReadmeConvention = TerracottaReadmeConvention) : AbstractProjectFile

Represents a README.md file in the project.

Exposes the full description and a short summary using a configurable ReadmeConvention. The default is TerracottaReadmeConvention.
