StateSourceConfig

data class StateSourceConfig(val projectDir: File, val settings: Map<String, String>)

Configuration passed to a StateSourceFactory when creating a StateSource.

Backends receive the project directory (so they can compute default paths) and a map of backend-specific settings supplied by the caller.
