# Terracotta

[Docs](https://beduality.github.io/terracotta/) | [Contributing](./CONTRIBUTING.md) | [MIT License](./LICENSE)

Declarative Minecraft project registry management tool. Sync project metadata to Modrinth, Hangar, and other registries from `terracotta.yml` or `build.gradle.kts`.
