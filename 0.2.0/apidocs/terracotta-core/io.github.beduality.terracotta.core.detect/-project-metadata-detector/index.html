interface ProjectMetadataDetector

Detects project metadata from the project directory.

Implementations may read platform-specific files (such as fabric.mod.json or mods.toml) and return any values they can infer. They should be registered as Java services via java.util.ServiceLoader.

Detectors receive a ProjectMetadataContext so that multiple detectors can share the file cache and the conventions used to interpret project files.
